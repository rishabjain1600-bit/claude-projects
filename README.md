# Claude Projects

A collection of projects built with [Claude](https://claude.ai) and the Anthropic API, exploring what's possible when AI acts as a capable, context-aware collaborator rather than a simple chatbot.

## Projects

| Project | Description |
|---|---|
| [job-search-tailoring](./job-search-tailoring/) | Paste a job description → get a perfectly tailored 1-page resume PDF and cover letter PDF, powered by Claude as an intelligent PM-style system |
